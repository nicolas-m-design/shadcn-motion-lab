# shadcn-motion-lab

A React + Vite interaction lab for exploring how AI changes the way interaction patterns are designed, specified, and implemented.

## V1 scope

- A gallery of 5 to 6 AI-native interaction patterns
- Real demos with lightweight notes, not deep documentation
- `shadcn/ui` as a pragmatic base layer, not the subject
- Motion included where it helps explain state and intent
- Companion material for a later article about workflow changes

## Planned setup

1. Use `shadcn/ui` primitives to build demo-ready shells quickly.
2. Implement a first set of patterns across observe, review, approve, and recover flows.
3. Keep pattern notes short: purpose, states, implementation, workflow learning.
4. Capture what AI accelerated, what required correction, and what was still manual.

## Notes

The current scaffold is intentionally light. It establishes a working surface for a portfolio-facing experiment archive rather than a polished library or design system.
